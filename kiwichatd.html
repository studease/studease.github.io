Preparation

For the Enterprise Edition, supply an interface to return a JSON object formatted as /data/userinfo.json.
While a client is connecting to upgrade protocol, it sends an identify upstream request, carrying channel and token params, to get the user info, which will decide whether the operation will be satisfied.

The Preview Edition is more like a stand-alone server. The user info, includes name, icon, role, and channel state could be present in params. However, this is not safe.

Run

Install OpenSSL at first. Then run: 

sudo ./start.sh
