Preparation

For Enterprise Edition, supply an interface to return a JSON object formatted as /data/userinfo.json.
While the client is connecting to upgrade protocol, we send an identify upstream request to get the user info, which decides whether the operation will be satisfied.

The Preview Edition won't send the identify upstream request.

Build

To build chatease-server, you need CMake 3.5 and or above. Then run: 

mkdir build 
cd build
cmake ..

Run

sudo ./chatease-server
